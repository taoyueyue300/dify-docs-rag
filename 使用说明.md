## 完整运行指南

### 1. 启动虚拟环境

**PowerShell（你当前用的）：**
```powershell
cd C:\Users\25352\Desktop\实习\dify-docs-rag
.venv\Scripts\Activate.ps1
```

如果报权限错误，先执行：
```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned
.venv\Scripts\Activate.ps1
```

成功后命令行前面会显示 `(.venv)`。

---

### 2. 可用的命令

| 命令                        | 作用                           | 什么时候用                     |
| --------------------------- | ------------------------------ | ------------------------------ |
| `python ingest.py`          | 文档入库（分块+向量化→FAISS）  | 首次运行 / 修改chunk参数后重跑 |
| `python retriever.py`       | 测试检索效果（3个示例问题）    | 验证检索是否正常               |
| `python chain.py`           | **命令行问答**（检索+LLM生成） | 日常使用                       |
| `streamlit run ui.py`       | **网页界面问答**               | 演示/截图用                    |
| `python app.py`             | FastAPI服务（API接口）         | 给其他程序调用                 |
| `python run_experiments.py` | 跑对比评测实验                 | 生成实验报告                   |

---

### 3. 典型使用流程

```powershell
# 进入项目 & 激活环境
cd C:\Users\25352\Desktop\实习\dify-docs-rag
.venv\Scripts\Activate.ps1

# 方式A：命令行问答
python chain.py
# 然后输入问题，如"如何本地部署Dify？"，输入 quit 退出

# 方式B：网页界面（推荐演示用）
streamlit run ui.py
# 自动打开浏览器 http://localhost:8501

# 方式C：API服务
python app.py
# 访问 http://localhost:8000/docs 查看接口文档
```

---

### 4. 注意事项

- **chain.py 和 ui.py 需要你的LLM API在线**（`127.0.0.1:8317`），确保你的代理服务在运行
- **retriever.py 不需要API**，纯本地检索测试
- **首次运行会下载模型**（bge-small-zh约90MB），之后走缓存
- 退出虚拟环境：输入 `deactivate`
